# 1. **包 package**

## 1.1. **概述**

 

​	CommonJS的包规范允许我们将一组相关的模块组合到一起，形成一组完整的工具。CommonJS的包规范由包结构和包描述文件两个部分组成。

​	包结构：用于组织包中的各种文件

​	包描述文件：描述包的相关信息，以供外部读取分析
